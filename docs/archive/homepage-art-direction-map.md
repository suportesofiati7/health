# Homepage art-direction map

This prototype translates the visual language found in `references/backgrounds/`,
`references/CTAS/`, `references/trustrows/`, `references/typography/`, and the
existing hero, CTA, botanical-background and footer systems. It preserves the
homepage's content, headings, links and semantic structure.
